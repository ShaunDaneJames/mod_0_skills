Class Computer

Attributes:
Computer_location (String)
IP_address (String)
POS_computer (Boolean)
power_on (Boolean)

Methods:
is_POS (returns the boolean if the computer is a point of sale computer or not.)
move_computer (modifies the string for computer name)
update_IP (updates the IP_address string)
toggle_power (changes the boolean value associated with power_on)
